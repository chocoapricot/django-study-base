このシステムは、サンプルデータを使って自由に機能を試すことができます。

以下の手順でサンプルデータを読み込み、管理者アカウントでログインしてください。

1. サンプルデータの読み込み

まず、システムの初期設定とサンプルデータの読み込みを行います。ターミナルで以下のコマンドを実行してください。

python manage.py load_sample_data
Note: このコマンドは、データベースを初期化し、あらかじめ用意されたサンプルデータを登録します。既存のデータはすべて削除されるのでご注意ください。

2. スーパーユーザーでのログイン

サンプルデータには、システムを管理するためのスーパーユーザーアカウントが含まれています。以下の情報でログインしてください。

  • メールアドレス: admin@example.com
  • パスワード: password
ログインページへ

3. 一般ユーザー（スタッフ）でのログイン

スタッフとしてログインし、スタッフ側の機能を確認することもできます。

  • メールアドレス: staff@example.com
  • パスワード: password

4. 一般ユーザー（クライアント）でのログイン

クライアント（顧客企業の担当者）としてログインし、クライアント側の機能を確認することもできます。

  • メールアドレス: client@example.com
  • パスワード: password
